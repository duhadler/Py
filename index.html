Welcome to mpFormulaPy.

The mpFormulaPy library is based on the Python runtime and the mpmath library, both of which implement multiprecision arithmetic.

Quick start under Windows.

System requirements: Windows (Desktop) with .NET Framework 4.x (Full).

Download the ZIP file and unzip it into a directory for which you have write access. Within the unzipped directory double-click on mpFormula.bat. This will start the Python Console of the SharpDevelop IDE. To confirm that mpFormulaPy is working, type the following within the Python Console:

>>> from mpFormulaPy import *
>>> pi

Still in the Sharpdevelop IDE, click on Tools -> mpFormulaPy Manual, to open the manual in your default pdf viewer. Within the manual, navigate to Getting Started - Tutorials, and follow the examples. You can copy and paste the Python code of the examples from the manual into the Python Console.

Manual

The manual is also available separately as pdf file: mpFormulaPy.pdf.

Quick start with Microsoft Excel

To use the mpFormulaPy functions from within Excel, you need to install the mpFormulaPy addin.
