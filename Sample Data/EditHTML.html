18 Inferential Statistics

This chapter describes functions for statistical tests and confidence intervals.

18.1 Procedures for 1 sample

Function mp_test_student_t_1sample(mean1 As mp_real, mean_ref As mp_real, stdev As mp_real, n As Long) As mp_table
This function calculates a result table for a t-test for 1 sample.

18.2 Procedures for 2 independent samples

Function mp_test_Student_t_2_samples(mean1 As mp_real, mean2 As mp_real, stdev1 As mp_real, stdev2 As mp_real, n1 As Long, n2 As Long) As mp_table
This function calculates a result table for a t-test for 2 independent samples.

18.3 Procedures for 2 correlated samples

Function mp_test_Student_t_2_samples_corr(mean1 As mp_real, mean2 As mp_real, stdev1 As mp_real, stdev2 As mp_real, n1 As Long, n2 As Long, correlation As mp_real) As mp_table
This function calculates a result table for a t-test for 2 correlated samples.

18.4 Procedures for k independent samples

Function mp_test_anova_1way(means As mp_vector, stdev As mp_vector, n As mp_vector) As mp_table
This function calculates a result table for a 1-way ANOVA.

18.5 Procedures for k correlated samples

Function mp_test_anova_1way_corr(means As mp_vector, stdev As mp_vector, n As mp_vector) As mp_table
This function calculates a result table for a 1-way ANOVA for correlated samples.

18.6 MANOVA and MANCOVA

Function mp_test_manova_1way(means As mp_matrix, CovarMatrix As mp_matrix, n As mp_vector) As mp_table
This function calculates a result table for a 1-way MANOVA.

18.7 Tests for covariance and correlation matrices

Function mp_test_canonical_correlation(CorrMatrix As mp_matrix, p As long q As long n As long) As mp_table
This function calculates a result table for a canonical correlation.
